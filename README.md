# 🧮 Калькулятор матричных операций

## 📋 О проекте

**Калькулятор матричных операций** — это десктопное приложение для выполнения различных операций над матрицами вещественных чисел. Программа позволяет выполнять базовые матричные операции, вычислять определители, решать системы линейных уравнений и находить обратные матрицы.

### Ключевые особенности

- ✅ **Базовые операции**: сложение, вычитание, умножение матриц, умножение на число, транспонирование
- ✅ **Вычисление определителя**: двумя методами (рекурсивный Лапласа для ≤10×10 и метод Гаусса для любых размеров)
- ✅ **Решение СЛАУ**: методом Гаусса с прямым и обратным ходом
- ✅ **Обратная матрица**: методом Гаусса-Жордана с автоматической проверкой A × A⁻¹ = I
- ✅ **Визуализация**: пошаговый лог всех вычислений с конкретными числами
- ✅ **Загрузка/сохранение**: поддержка текстовых файлов (.txt, .csv)
- ✅ **Случайные матрицы**: генерация случайных чисел от -10 до 10

## 🎯 Реализованные алгоритмы

| Алгоритм | Описание | Сложность |
|----------|----------|-----------|
| **Рекурсивный определитель (Лаплас)** | Разложение по первой строке | O(n!) |
| **Определитель (Гаусс)** | Приведение к треугольному виду | O(n³) |
| **Решение СЛАУ (Гаусс)** | Прямой и обратный ход | O(n³) |
| **Обратная матрица (Гаусс-Жордан)** | Расширенная матрица [A\|E] → [E\|A⁻¹] | O(n³) |
| **Умножение матриц** | Классический алгоритм | O(n³) |
